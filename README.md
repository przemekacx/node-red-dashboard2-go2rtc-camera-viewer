# node-red-dashboard2-go2rtc-camera-viewer
Live low-latency WebRTC camera viewer for Node-RED FlowFuse Dashboard 2

## Features

- 🚀 WebRTC low latency streaming
- 📱 Mobile friendly UI
- 🏠 Home automation ready
- 🎥 Multiple cameras support
- 🐳 Docker based
